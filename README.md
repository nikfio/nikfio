# About me

Mechatronic Engineer making software to enhance the automation of any machine.  
I believe any machine can implement an adaptive layer to optimize and improve all its features throughout its lifecycle.  
I am a Reinforcement Learning passionate, and enthusiastic to explore its application in the physical world.

# Projects

## Forex-data 

A data layer that convolves usage of multiple data providers to have historical and realtime data of any forex ticker.

[Github repository](https://github.com/nikfio/forex-data)

## LSTM on ASCII maps

Application of an LSTM neural network to execute the path planning task on simplified virtual maps realized with ASCII characters.
This work was developed during my master degree studies.

[Github repository](https://github.com/nikfio/LSTM_ASCII_maps)

## LSTM on iRobot Create 2

Application of LSTM neural network as a path planner for mobile robots.
In this project, the model was used on the iRobot Create 2.

[Github repository](https://github.com/nikfio/LSTM_on_Create2)
